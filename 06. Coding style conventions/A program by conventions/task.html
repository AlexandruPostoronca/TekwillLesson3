A program by conventions

Here are parts of four programs. Which one follows the discussed code style conventions?

a)

public static void main(String[] args) {
        System.out.println("The more spaces the better!");
}

b)

public static void main(String[] args)
{
    System.out.println("Braces should be on the same level!");
}

c)

public static void main(String[] args) {
System.out.println("We do not need extra spaces!");
}

d)

public static void main(String[] args) {
    System.out.println("Four spaces are enough!");
}
